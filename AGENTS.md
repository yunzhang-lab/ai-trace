# ai-trace-open Agent Guide

> 本文件为 AI Agent 专属行为准则与上下文约定。
> 仓库结构、各模块入口与阶段定义请统一参考 [README.md](/README.md)。

## 1. 协作边界

1. 禁止将真实路径、密钥、原始日志提交到公开仓。

2. 私有数据默认位于：
2.1 `~/.ai-trace/`

3. 讨论记录遵循：
3.1 `discussions/README.md`
3.2 采用 Append-Only 方式增量记录

4. 代码中禁止硬编码绝对物理路径与个人密钥。

## 2. 输出风格

1. 默认使用总分结构。

2. 先给结论，再给分类展开。

最好单次300字以内
3. 编号统一使用：
3.1 `1`
3.2 `1.1`
3.3 `1.2`

4. 保持简短，避免长段铺陈。

5. 非必要不复述背景，不重复用户原话。

6. 当用户要求“只分析”时，优先按以下结构回答：
6.1 已完成  
6.2 未完成  
6.3 不一致  
6.4 建议动作

7. 除非用户明确要求展开，否则默认输出短答版。

8. 单次回复优先控制在 10-20 行内；内容较多时先给摘要版。
