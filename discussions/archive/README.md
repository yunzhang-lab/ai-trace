- 已被新版本路线图或总纲替代的提案

归档规则：

1. 归档不等于删除，文件仍需保留元数据与原始讨论内容。
2. 被归档文件必须在 `discussions/README.md` 的索引表中保留记录，并将状态标为 `archived`。
3. 若归档文件仍被新文件引用，应在新文件或索引表中注明“已归档但可追溯”。
