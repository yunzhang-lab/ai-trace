本目录只承载阶段规则，不承载进度看板、具体计划或稳定规范。阶段可以引用 `PROGRESS.md` 中的 `Step`，但不直接替代 `Step` 的进度表达。

## 1. 职责

1. 定义阶段边界。
1.1 每阶段允许解锁什么

1.3 阶段之间如何衔接
1.4 每阶段覆盖哪些 `Step`
1.5 每阶段把相关 `Step` 收口到什么程度

2. 不承载具体进度。
2.1 阶段与子能力进度进入 [PROGRESS.md](/PROGRESS.md)
2.2 未确认计划进入 [drafts/plans/](/drafts/plans/)
2.3 稳定规范进入 [skill/ai-trace/specs/](/skill/ai-trace/specs/)
